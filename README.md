# Tampermonkey 脚本集合

这个仓库包含了一系列用于 Tampermonkey 浏览器扩展的实用脚本，旨在增强特定网站的功能和用户体验。


## 脚本列表

| 项目名称 | 路径 | 
| --- | --- | 
| RZLowCode开发工具箱 | [RZLowCode开发工具箱 0.5.0 ](https://github.com/fanji-jared/tampermonkey/tree/main/RZLowCode) | 
## 脚本介绍

### CodeCV简历保存打印脚本

在 CodeCV 网站右上角添加一个蓝色小球按钮，点击后自动进入打印界面，可以选择导出为PDF，实现简历的保存和打印功能。

### Duome Stories 音频控制器

优化 Duome.eu/stories 网站的音频体验，提升英语学习效率：
- 支持音频预加载
- 快捷键控制播放
- 播放句子高亮显示
- 快捷键设置存储
- 键盘监听优化

## 使用方法

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击上方表格中的脚本链接
3. 在打开的页面中点击「安装」或「Raw」按钮
4. 访问对应网站，脚本将自动生效
