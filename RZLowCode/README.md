# 介绍

为RZ低代码平台添加的一个开发工具箱，提供一些实用的功能，帮助开发者提高开发效率。

## 已开发功能

| 版本 | 功能名称 | 介绍 |
| --- | --- | --- |
| v0.5.0 | 字段列宽计算 | 自动计算table的th最小宽度，且可预览和调整，一键生成配置字符串 |
| v0.6.0 | 设置模板代码框高度 | 优化现有“应用设计”，支持通过拖拽方式直观调整 CodeMirror 编辑器的代码框高度，适配不同长度的 “模板代码” 编写场景，提升编辑效率 |

## 使用方法

### 基本使用流程（以"字段列宽计算"为例）

1. 打开工具箱，进入"字段列宽计算"应用
    - 在低代码平台页面中，点击右下角的蓝色🔧图标
    - 工具箱面板会展开显示所有应用
    - 点击"字段列宽计算"应用图标

2. 扫描表格
    - 点击"扫描表格"按钮
    - 脚本会自动扫描页面中的所有表格
    - 鼠标悬停在表格列表项上可预览对应表格

3. 选择目标表格
    - 从扫描结果列表中选择要操作的表格
    - 选中的表格会显示5秒红色边框作为确认

4. 调整列宽
    - 查看自动计算的列宽值
    - 使用+/-按钮或直接输入调整各列宽度
    - 操作列宽度会自动计算最小避免换行值

5. 应用修改
    - 点击"应用修改"按钮将宽度应用到表格
    - 修改的单元格会显示5秒红色边框作为反馈

6. 生成格式字符串
    - 点击"生成字符"按钮生成平台所需的格式字符串
    - 使用"复制"按钮快速复制到剪贴板
